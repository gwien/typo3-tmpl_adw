11.3 Links
==========

Externe Links
-------------
Bei externen Links gibt es die Möglichkeit, diesen in einem neuen Fenster/Tab öffnen zu lassen. Dazu muss die folgende Einstellung vorgenommen werden:

.. figure:: ../../Images/StyleGuideline/ExterneLinks.png
	:width: 400px
	:alt: Dateiablage des TYPO3-Backend

Links auf Dokumente
-------------------
Bei Links auf Dokumenten sollte das Dateiformat und die Dateigröße in Klammern nach dem Link mit angegeben werden.

**Beispiel:** Programmheft (PDF, 300 kb)
