Positionierung von Bildern¶

In der rechten Spalte¶

Bild und Bildunterschrift werden immer linksbündig dargestellt. Die Bildgröße wird per default auf die gesamte Breite der Spalte angepasst.

In der Content-Spalte¶

In der mittleren Inhaltsspalte stehen verschiedene Möglichkeiten zur Positionierung von Bildern zur Verfügung. - Bild als eigenes Inhaltselement einfügen. In der Grundeinstellung wird das Bild auf die gesamte Spaltenbreite skaliert. - Bild mit Text: Das Bild ist oben links platziert und der Text fließt um das Bild. Die Bildgröße wird automatisch angepasst.

Bei beiden Optionen können die Bildbreiten auch selbst angepasst werden, wovon aus ästhetischen Gründen abgeraten wird.
